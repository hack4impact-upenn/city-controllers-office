Search Tips

Refer to the following tips when searching for contracts.

Searching by City Contracts

  • Leave all fields blank to show all the contracts on record
  • Type in your the Vendor name you are searching. It does not have to be the exact name. If you do not know the vendor name, leave blank.
  • Type in any keywords associated with this contract. Keyword search looks through vendor name, department name, contract type, advertising status, and description. This will search using the entire entry into the Keyword field, rather than performing separate searches for different words.
  • You can leave one end of the date or amount range unbounded. For example, if you want to search for all contracts above $50,000 you can put 50000 as minimum and leave maximum blank.
